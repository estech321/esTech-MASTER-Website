Blog / About This Site

Welcome to the esTech website! On this website, I create tutorials, blog posts, extra resources to go with a YouTube video, or even showcases of projects. This website is associated with a YouTube channel named esTech, in which I post videos on all things technology!

How I Built This Website

To create this website, I started by finding a template I liked from HTML5Up.net, my favorite website template source. I like HTML5Up because all of their content is under a simple Creative Commons license, keeping copyright simple. Their designs are fantastic, as well. I scrolled through their list of templates until I decided on Editorial. From there, I built custom pages using VS Code.

How It Works

This website uses the font Roboto Slab, with icons from Font Awesome. I made it in Visual Code Studio and host it using GitHub Pages.
